# michaelpyon.com

Personal portfolio site. Built with React, Vite, and Tailwind CSS.

## Projects

- **[ShooterDigest](https://shooter.michaelpyon.com)** — Weekly intelligence briefing for competitive FPS games
- **[Air Composer](https://air-composer.michaelpyon.com)** — Hand-tracking theremin and talk box in the browser

## Stack

React · Vite · Tailwind CSS v4 · Vercel

## Run locally

```bash
npm install
npm run dev
```
