# BlockScore

NYC apartment hunting intelligence. Aggregates StreetEasy data with walk scores, subway proximity, noise complaints (311 data), and restaurant density for any block in Brooklyn & Manhattan.

## Stack

- React 19
- Vite
- Tailwind CSS v4

## Data Sources

- **StreetEasy** — Median rents, inventory, days on market, price/sqft
- **Walk Score** — Walkability ratings
- **NYC 311** — Noise complaint counts and breakdown by type
- **Google Places** — Restaurant counts, cuisine types, density

## Development

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
```
